What is this?

This is the complementary web-page for Eurographics 2022 State-of-The-Art Report (under review): Neural Fields in Visual Computing and Beyond. We hope to provide a comprehensive yet interactive database of all papers using the coordinate-based neural network technique. For more in-depth discussions, please visit our review paper ( Bibtex ).

Bibtex for our paper: @article{xie2021neuralfield, TITLE = {Neural Fields in Visual Computing and Beyond}, AUTHOR = {Yiheng Xie and Towaki Takikawa and Shunsuke Saito and Or Litany and Shiqin Yan and Numair Khan and Federico Tombari and James Tompkin and Vincent Sitzmann and Srinath Sridhar}, BOOKTITLE = {ArXiv Pre-print}, YEAR = {2021} }

For authors, please visit the info page to find instructions on how to add your paper to the database, or edit your existing entry.

Contributors

Features

Filtering

Users can filter out disired papers by applying one or multiple filters on papers' titles, authors, keywords, or publication dates.

Building a timeline

A scrollable and zoomable timeline can be built given a set of papers. Users can use this feature to check the trajectory of how a certerin technique has evolved over time.

Citation graph

Citation graph offers an intuitive way to check citation relations given a single paper or a set of papers.
