About this website

How to submit a new paper or update an existing paper?

First, please note that papers can only be submitted / updated by their authors. All submissions / updates will be reflected in the website upon administrators' approval.

To submit a new paper to our database, please follow the steps below:

  • Step 1: Fill this Google form with relevant information about your paper.
  • Step 2: Clone our Github Repository locally.
  • Step 3: Go to this spreadsheet, copy the row corresponding to your paper.
  • Step 4: Paste that row at the bottom of file "sitedata/papers.csv".
  • Step 5: Commit changes to your cloned repository and create a pull request with title "Submitting paper..."
    • .

To update an existing paper, please follow the steps below:

  • Step 1: Fill this Google form with information for updating your paper.
  • Step 2: Clone our Github Repository locally.
  • Step 3: Go to this spreadsheet, copy the row corresponding to your paper.
  • Step 4: Paste that row at the bottom of file "sitedata/papers.csv".
  • Step 5: Commit changes to your cloned repository and create a pull request with title "Updating paper...".
