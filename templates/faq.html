What is this?

This is the complementary web-page for Eurographics 2022 State-of-The-Art Report (under review): Neural Fields in Visual Computing and Beyond.

We aim to provide a comprehensive yet interactive database of all papers using the coordinate-based neural network technique.

For more in-depth discussions, please visit our review paper.

Features

Users can filter out disired papers by applying one or multiple filters on papers' titles, authors, keywords, or publication dates.

A scrollable and zoomable timeline can be built given a set of papers. Users can use this feature to check the trajectory of how a certerin technique has evolved over time.

Citation graph offers an intuitive way to check citation relations given a single paper or a set of papers.

Submit a new paper or update an existing paper

To submit a new paper or update an existing entry, please fill out this Google form. Papers can only be submitted or updated by their authors. Changes will be reflected in the website upon administrators' approval.

If you would like to see the changes reflected immediately, please follow these additional steps:

  • Step 1: Clone our Github Repository locally.
  • Step 2: Go to this spreadsheet, copy the row you just entered (first row).
  • Step 3 (new paper): Append the row to the top (row 1) in file "sitedata/papers.csv" in your cloned local repository.
  • Step 3 (update existing): Replace the existing row (corresponding to your paper) in file "sitedata/papers.csv" with the row you copied.
  • Step 4: Commit changes to your cloned repository and create a pull request.

Disclaimer

Data accuracy

Up until the release date (Oct.29, 2021), all ~250 data entries were imported manually by our contributers. Please be patient with our mistakes. We encourage all authors to verify that the data corresponding to their papers are accurate by using our submission/editing tool (see instructions below). We also encourage all authors to update the venue, once an arXiv pre-print is accepted by a peer-reviewed venue.

Peer-review criteria

In this website, we include all published results, which includes both peer-reviewed papers and arXiv pre-prints. We recognize the importance and rigor of peer-review, so we have included the venue in each paper's landing page. When searching, you may filter for peer-reviewed papers only by selecting the corresponding check box.
