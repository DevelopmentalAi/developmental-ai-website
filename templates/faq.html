What is this?

This is the complementary web-page for arXiv pre-print : Neural Fields in Visual Computing and Beyond.

We aim to provide a comprehensive yet interactive database of all papers using the coordinate-based neural network technique.

For more in-depth discussions, please visit our review paper.

Features

Users can filter out desired papers by applying one or multiple filters on papers' titles, authors, keywords, or publication dates.

A scrollable and zoomable timeline can be built given a set of papers. Users can use this feature to check the trajectory of how a certain technique has evolved over time.

Citation graph offers an intuitive way to check citation relations given a single paper or a set of papers.

Disclaimer

Data accuracy

Up until the release date (Oct.29, 2021), all ~250 data entries were imported manually by our contributers. Please be patient with our mistakes. We encourage all authors to verify that the data corresponding to their papers are accurate by using our submission/editing tool (see instructions below). We also encourage all authors to update the venue, once an arXiv pre-print is accepted by a peer-reviewed venue.

Peer-review criteria

In this website, we include all published results, which includes both peer-reviewed papers and arXiv pre-prints. We recognize the importance and rigor of peer-review, so we have included the venue in each paper's landing page. When searching, you may filter for peer-reviewed papers only by selecting the corresponding check box.
